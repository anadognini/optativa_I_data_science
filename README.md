# Practical Assignment 1
The purpose of this assignment was to operationalize core data preprocessing concepts through a structured workflow. The chosen dataset for this was: [Sleep Health and Lifestyle Dataset](https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset/data). 

The tasks of the assignment were:
- To provide a brief but analytically grounded justification for the selected dataset
- To develop or refine a structured data dictionary
- To do initial data cleaning and preparation

## Files
- Google Colab Notebook: [`optativa_I_practical_assignment_1.ipynb`](https://colab.research.google.com/drive/1rV7SjYRdRc1mllIBagVqr8gYPJQzulk_):
    - Short report containing an overview of the chosen dataset
    - Data dictionary summary
    - Simple data cleansing
- File containing the complete data dictionary: `data_dictionary.csv`
